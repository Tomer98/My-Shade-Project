# ☀️ Smart Shade Project

A smart campus shading management system for HIT. 
This project provides a real-time dashboard to monitor and automate window shades based on temperature, light intensity, and weather conditions.

## Features
* 🗺️ **Campus Map:** Interactive map with draggable sensors.
* 🧠 **Smart Algorithm:** Real-time decision making based on live data.
* 📅 **Task Scheduler:** Automate opening/closing shades.
* 👥 **User Management:** Role-based access control (Admin, Maintenance, Planner).
* 🚨 **Real-time Alerts & Logs:** Powered by Socket.io.

## Tech Stack (Client)
* React + Vite
* Axios
* Recharts (Data Visualization)
* Socket.io-client
